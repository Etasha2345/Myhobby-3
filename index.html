My Favourite Hobby

Coding

I love to code

My favorite hobby is Coding. Coding is the process of using a programming language to get a computer to behave how you want it to. Every line of code tells the computer to do something, and a document full of lines of code is called a script. Each script is designed to carry out a job. Coding is a list of step-by-step instructions that get computers to do what you want them to do. Coding makes it possible for us to create computer software, games, apps and websites. Coders, or programmers, are people who write the programmes behind everything we see and do on a computer. Simply put, coding is used for communicating with computers. People use coding to give computers and other machines instructions on what actions to perform. Further, we use coding to program the websites, apps, and other technologies we interact with every day.
I enjoy coding as it gives me pleasure of thinking logically and help me in thinking out of box. I enjoy imagine the things and create on digital media through coding.
